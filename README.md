# EdgeProTrader

**EdgeProTrader** は、裁量トレードにおける「計画 → エントリー → 記録 → 分析」までのライフサイクル全体を支援するトレードエコシステムです。  
本プロジェクトは、MetaTrader（MQL4/MQL5）をベースに、トレーダーの意思決定と振り返りを構造化・再利用可能にすることを目指しています。

---

## 🚀 プロジェクト概要

本リポジトリでは以下の要素を段階的に開発・管理しています：

- **Entry機能**：成行発注、建値移動、TP/SL操作などを支援するパネルUI（MQL4実装）
- **Planning機能**：チャート上での戦略可視化・記録（MQL5構想）
- **Journal機能**：Notion連携による戦略・スクリーンショット一括保存（Python連携）
- **Analytics機能**：トレードログの可視化・再分析（AI連携構想あり）

> 詳細は `/docs/EPT_Project_v2.1.md` にて構想を記述しています。

## 🛠 開発方針・原則

- **OOP徹底**：1クラス1責務、View/Logic/Tradeの三層分離
- **ドキュメントファースト**：すべての仕様・設計は`.md`で管理
- **段階開発**：Entry → Planning → Journal → Analyticsの順に拡張
- **MQL5移植を前提とした構造設計**
